# Dotfiles

My personal terminal setup.

## Setup

```sh
./setup.sh
```

## Formatting

Shell files (zshrc, aliases, functions) are formatted with [shfmt](https://github.com/mvdan/sh):

```sh
./format.sh
```

For vimrc, open it in Vim and run `gg=G` to re-indent the file.

## Dependencies

- [Homebrew](https://brew.sh)
- [Oh My Zsh](https://ohmyz.sh)
- [mise](https://mise.jdx.dev)
